Installation Instructions

Extract Resource Pack

Step 01

Locate the downloaded autocommit.zip file on your computer and extract it. This generates the folder containing the manifest.json and pre-compiled scripts.

required
01

Open Extensions Console

Step 02

Open Google Chrome or Microsoft Edge and navigate to the Extensions dashboard by typing chrome://extensions/ (or edge://extensions/) in the address bar.

browser-console
02

Toggle Developer Mode

Step 03

Locate the Developer Mode switch in the top-right corner of the Extensions page and toggle it **ON**. This allows you to load custom extensions locally.

system-setting
03

Load Unpacked Extension

Step 04

Click the Load unpacked button in the top-left corner. Select the extracted folder containing manifest.json to install and run the extension.

final-step
04

Extension Features

Scraper

LeetCode Auto-Sync

Monitors LeetCode outcome status. Instantly extracts the accepted solution code, language file extension, and difficulty badges when a test run completes.

Scraper

Codeforces Sync

Parses submission viewer detail pages. Scrapes rating scores, problem tags, programming language, and the full code body automatically.

Git Core

Conflict Protection

Retrieves the existing file SHA hash before pushing to GitHub to prevent API overwrite conflicts, keeping your commit log clean and unbroken.

Options

Folder Layouts

Customizable structures. Group your repository files by platform folders (leetcode/), by difficulty (Easy/, Medium/), or upload flat files at the root level.

Manual

Manual Push Panel

A manual paste fallback panel. Easily copy-paste code, titles, and select difficulty labels to push solutions from other unsupported sites.

Security

100% Client-Side Privacy

Direct GitHub REST API wrapper. Your access tokens and push history are saved securely on your local chrome storage sandbox. No proxy servers used.
