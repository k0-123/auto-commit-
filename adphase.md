# Advanced Phases

> These phases come after the core functionality is solid.
> They add power-user features and polish.

---

## Phase 7: Keyboard Shortcuts & Power User ✅
> **Goal:** Fast interactions without mouse

### Tasks
- [x] Global keyboard shortcuts (e.g., `Ctrl+Shift+T` to open task)
- [x] Command palette in popup (Cmd+K style)
- [x] Inline task editing via keyboard
- [x] Quick-add with natural language (e.g., "Buy groceries tomorrow #shopping")
- [x] Shortcut customization in options page

### Deliverables
- Power users can navigate entirely with keyboard
- Natural language task creation
