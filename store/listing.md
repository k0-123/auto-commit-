# Productivity Assistant — Chrome Web Store Listing

## Name
Productivity Assistant

## Summary (max 80 chars)
Tasks, focus timer, and quick capture — all from your browser toolbar.

## Description
Productivity Assistant is a local-first Chrome extension that helps you stay
organized and focused without leaving your browser.

**Plan**
Create tasks in seconds with natural-language quick add — try
“Call mom tomorrow #family !high”. Tasks support priority, due dates,
categories, and tags.

**Focus**
A built-in Pomodoro timer keeps you on track, with an optional website
blocker during focus sessions and daily/weekly focus stats.

**Capture**
Highlight any page and capture it to your task list via the right-click menu,
or grab a screenshot and notes without breaking flow.

**Insights**
Track completion rate, streaks, time-per-category, and an activity heatmap.
Export your data to CSV or JSON anytime.

**Private by default**
Everything is stored locally with Chrome Storage. Optional AI features run
offline with smart heuristics; connect your own API key only if you want
LLM-powered task breakdown. Sync and third-party integrations are modular and
opt-in.
