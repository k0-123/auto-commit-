# AutoCommit — Store Listing Details

## Description
AutoCommit is the ultimate companion extension for competitive programmers. It automatically tracks your solution submissions on LeetCode and Codeforces, extracts the accepted code, and pushes it directly to your selected GitHub repository in a single click.

Keep your personal coding portfolio up-to-date and showcase your daily algorithm progress with absolutely zero friction!

### Key Features:
* ⚡ **Instant Solution Interception**: Auto-detects when you receive an "Accepted" verdict on LeetCode or Codeforces.
* 📂 **Smart Folder Layouts**: Organize your repository by platform (e.g. `leetcode/TwoSum.py`) or by difficulty level (e.g. `Easy/TwoSum.py`).
* 🛡️ **Conflict-Free Commits**: Automatically fetches existing file SHAs before pushing to GitHub, preventing API errors and preserving your Git history.
* 📝 **Custom Commit Messages**: Automatically drafts descriptive commits (e.g., `feat: solve Two Sum (leetcode)`), with option to fully customize before pushing.
* 🛠️ **Manual Sync Panel**: Paste and push custom code, titles, and difficulty labels for other platforms.
* 🔒 **Private & Secure**: Connects directly to GitHub via your personal access token. All keys and data are saved locally inside your browser's secure sandbox.
